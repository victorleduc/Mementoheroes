## Want to try locally? ##

```bash
#cloning code
git clone https://github.com/LudovicLanchon/Mementoheroes.git
cd Mementoheroes

#install dependencies
npm intall
#start debug server
npm start
```

Now, view the demo at [http://localhost:4200](http://localhost:4200)
